# ZigZag

![Screenshot](./screenshots/screenshot.png)

Stay on the wall and zigzag as far as you can!
Just tap the screen to change the ball’s direction.  
One wrong move and you’re off the edge!
How far can you make it?

## Technologies

- React
- Three.js
- React Three Fiber
- Zustand
